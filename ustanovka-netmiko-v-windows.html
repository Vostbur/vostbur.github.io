Установка netmiko в Windows

Модуль netmiko дает красивый интерфейс поверх paramiko и telnetlib для подключения к сетевым устройствам по SSH или Telnet.

Чтобы не связываться с установкой кучи всего от Microsoft для исправления ошибки "Microsoft Visual C++ 14.0 is required":
1. Ставим Miniconda
2. Выполняем
conda install paramiko
pip install netmiko
