  • Nornir - фреймворк на Python, альтернатива Ansible для автоматизации конфигурирования сетевых устройств.
  • NetBox - DCIM/IPAM open source (Django Python) система от DigitalOcean. Есть готовая сборка в Docker-контейнер. Кроме web-интерфейса есть API. Ознакомительная статья.
    Я вижу такой алгоритм работы: инвентари-файл с описанием оборудования сети парсится в json и отдается NetBox, всё, что предлагает NetBox для конфигурирования адресации, получается по API и разносится по устройствам ansible или аналогами.
